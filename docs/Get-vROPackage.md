# Get-vROPackage

## SYNOPSIS
Get vRO Packages

## SYNTAX

### All (Default)
```
Get-vROPackage
```

### Name
```
Get-vROPackage -Name <String>
```

## DESCRIPTION
Get vRO Packages

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-vROPackage
```

### -------------------------- EXAMPLE 2 --------------------------
```
Get-vROPackage -Name 'com.vmware.library.powershell'
```
