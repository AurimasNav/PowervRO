# Get-vROUser

## SYNOPSIS
    
Returns the solution user and whether the current user has admin rights as well as its member groups

## SYNTAX
 Get-vROUser [<CommonParameters>]    

## DESCRIPTION

Returns the solution user and whether the current user has admin rights as well as its member groups


## INPUTS

None

## OUTPUTS

System.Management.Automation.PSObject.

PS C:\>Get-vROUser
