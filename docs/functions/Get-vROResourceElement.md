# Get-vROResourceElement

## SYNOPSIS
Retrieves a list of all resource elements

## SYNTAX

### All (Default)
```
Get-vROResourceElement
```

### Id
```
Get-vROResourceElement -Id <String[]>
```

## DESCRIPTION
Retrieves a list of all resource elements

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-vROResourceElement
```

### -------------------------- EXAMPLE 2 --------------------------
```
Get-vROResourceElement -Id f2193849-89e9-4136-8607-526eb196ee4c
```
